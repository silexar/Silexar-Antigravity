## 🔧 **SI AÚN SE VE MAL:**

### **Opción 1: Forzar recarga completa**
```bash
# En la terminal
Ctrl + C  # Detener servidor

# Limpiar todo
rmdir /s /q .next
rmdir /s /q node_modules
npm install
npm run dev
```

### **Opción 2: Verificar en navegador**
1. Abre **Chrome** o **Firefox**
2. Presiona `Ctrl + Shift + R` (recarga forzada)
3. O presiona `F12` → pestaña **Network** → marcar "Disable cache"

### **Opción 3: Verificar puerto**
Si el puerto 3000 está ocupado:
```bash
npm run dev -- -p 3001
# Luego ir a: http://localhost:3001/command-center
```
