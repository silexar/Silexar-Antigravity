## 🚀 **DEPLOYMENT INSTRUCTIONS**

### 1. Environment Setup
```bash
# Clone repository
git clone https://github.com/silexar/pulse-quantum.git
cd pulse-quantum

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Edit .env.local with production values
```

### 2. Production Build
```bash
# Build for production
npm run build

# Test production build locally
npm start
```

### 3. Deploy to Vercel (Recommended)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy to production
vercel --prod
```

### 4. Docker Deployment
```bash
# Build Docker image
docker build -t silexar-quantum .

# Run container
docker run -p 3000:3000 silexar-quantum
```
